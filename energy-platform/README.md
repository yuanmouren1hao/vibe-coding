# 新能源虚拟电厂管理平台

基于 React + TDesign + Recharts 构建的新能源虚拟电厂管理平台可视化系统。

## 功能特性

### 📊 核心功能
- **实时监控**: 实时展示太阳能、风能、水电发电量
- **地图分布**: 中国地图展示全国电站分布情况
- **储能管理**: 监控电池、水电、热储存等储能设备状态
- **效率分析**: 仪表盘展示运行效率
- **数据统计**: 能源结构饼图、历史数据柱状图
- **天气预报**: 未来48小时天气及发电预测

### 🎨 技术栈
- **前端框架**: React 18 + TypeScript
- **UI组件库**: TDesign React v1.12.0
- **图表库**: Recharts v2.10.0
- **样式方案**: TailwindCSS v3.4.17
- **构建工具**: Vite v5.0.8

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 📊 数据说明

### Mock数据包含
- **实时发电数据**: 24小时太阳能、风能、水电发电量
- **电站分布数据**: 全国26个电站位置、容量、状态信息
- **储能设备数据**: 3种储能设备（电池、水电、热储存）状态
- **历史运行数据**: 12个月发电量统计
- **天气预报数据**: 未来48小时天气及发电预测
- **能源结构数据**: 各类能源占比

### 数据更新
- 实时发电量每3秒更新一次
- 运行效率自动波动模拟
- 所有数据均为真实场景模拟

## 🎨 设计特点

### 视觉风格
- **深色科技风**: 深蓝色渐变背景，营造科技感
- **霓虹色彩**: 青色、绿色、黄色等霓虹色点缀
- **玻璃态效果**: 半透明卡片配合背景模糊
- **动态效果**: 图标脉冲动画、数据实时更新

### 交互设计
- **响应式布局**: 支持PC、平板、手机等多端适配
- **悬浮提示**: 鼠标悬浮显示详细信息
- **颜色编码**: 不同能源类型使用不同颜色区分
- **状态指示**: 实时显示设备运行状态
