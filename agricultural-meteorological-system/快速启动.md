# 🚀 快速启动指南

## 方法一: 使用启动脚本 (推荐)

直接双击运行项目根目录下的 `start.bat` 文件,脚本会自动:
1. 安装所有依赖
2. 启动开发服务器
3. 自动打开浏览器

## 方法二: 手动命令行启动

### 1. 安装依赖
```bash
cd c:/UGit/github/codebuddy-playground/agricultural-meteorological-system
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
浏览器自动打开或手动访问: `http://localhost:5173`

## ⚠️ 常见问题

### Q: 端口 5173 被占用怎么办?
A: Vite 会自动尝试下一个可用端口(5174, 5175...)

### Q: 安装依赖很慢?
A: 可以使用国内镜像:
```bash
npm install --registry=https://registry.npmmirror.com
```

### Q: 看不到地图?
A: 地图使用的是简化的 GeoJSON 数据,已内置在项目中,无需额外配置

## 📱 功能导航

启动后您可以:

1. **查看总览页** (`/overview`)
   - 查看上海10个农场的实时气象数据
   - 在地图上点击农场标记查看详情
   - 查看区域平均指标和预警信息

2. **对比农场** (`/compare`)
   - 选择2-4个农场进行对比
   - 查看雷达图和时间序列图

3. **趋势分析** (`/trend`)
   - 选择农场和时间范围
   - 查看历史数据趋势
   - 导出数据

4. **农场详情** (点击农场名称)
   - 查看8个仪表盘数据
   - 查看24小时预测
   - 获取适种建议

## 🎯 测试建议

1. 点击侧边栏的"刷新数据"按钮,查看数据动态更新
2. 在总览页点击不同农场标记,跳转到详情页
3. 在对比页选择多个农场,查看雷达图对比
4. 在趋势页选择不同时间范围,观察数据变化

享受您的数据可视化之旅! 🌾
